# 基本概念
## 平衡解
在微分方程中，**平衡解 (Equilibrium Solution)** 也被称为**驻定解 (Stationary Solution)** 或**常数解 (Constant Solution)**。
### 概念解释
对于一个形如 $\frac{dy}{dt} = f(y)$ 的**自治一阶常微分方程 (Autonomous First-Order ODE)**，如果存在一个常数 $y = c$ 使得：
$$f(c) = 0$$

那么 $y(t) = c$ 就是这个微分方程的一个平衡解。

---
### 主要特点
1. **常数解:** 平衡解是一个**不随时间 $t$ 变化的**解。如果系统处于平衡态 $y=c$，它将永远保持在这个状态。
2. **导数为零:** 平衡解的定义意味着 $\frac{dy}{dt} = 0$，即系统的状态不发生变化。
3. **几何意义:** 在相线图 (Phase Line) 上，平衡解对应于 $f(y)$ 函数图与 $y$ 轴的交点。
4. **稳定性:** 平衡解的**稳定性 (Stability)** 非常重要，它决定了附近的解会如何演化：
    - **稳定 (Stable) 或 渐近稳定 (Asymptotically Stable):** 附近的解会随着时间趋近于平衡解。
    - **不稳定 (Unstable):** 附近的解会随着时间远离平衡解。
    - **半稳定 (Semi-stable):** 一侧的解趋近，另一侧的解远离。

## 偏差范数
**偏差范数 (Deviation Norm)** 是在微分方程和控制理论中，用来衡量系统的**解 $x(t)$** 与其**平衡解 $x^*$** 之间距离或偏离程度的数学工具。
### 1. 基本概念
当讨论微分方程的稳定性时，我们通常假设平衡解为零解 $x^*=0$，因此偏差范数 $\lVert x(t) - x^* \rVert$ 简化为 $\lVert x(t) - 0 \rVert = \lVert x(t) \rVert$。
您图片中提到的零解 $x=0$ 的稳定性定义，就是在使用这种简化的偏差范数：
- **初始偏差范数:** $\lVert x_0 \rVert$
    - 表示**初始状态** $x_0$ 偏离平衡点 $x=0$ 的距离。
- **解的偏差范数:** $\lVert x(t) \rVert$
    - 表示**在时间 $t$ 时的状态** $x(t)$ 偏离平衡点 $x=0$ 的距离。

### 2. “范数” (Norm) 的意义
**范数** $\lVert \cdot \rVert$ 是一个数学概念，它赋予向量空间中的向量（在这里是状态向量 $x$）一个“长度”或“大小”，用来度量向量空间中两点之间的距离。
对于一个 $n$ 维向量 $x = (x_1, x_2, \dots, x_n)^T$，常用的范数有以下几种：

| **范数名称**                     | **记号 (定义)**                                                                                    | **应用场合**           |
| ---------------------------- | ---------------------------------------------------------------------------------------------- | ------------------ |
| **欧几里得范数 ($L_2$ 范数)**        | $\lVert x \rVert_2 = \sqrt{x_1^2 + x_2^2 + \dots + x_n^2}$                                     | 最常用，对应几何空间中的直线距离。  |
| **曼哈顿范数 ($L_1$ 范数)**         | $$\lVert x \rVert_1 = \sum_{i=1}^{n} \|x_i\| = \|x_1\| + \|x_2\| + \dots + \|x_n\|$$           | 曼哈顿范数定义为其所有分量绝对值的和 |
| **切比雪夫范数 ($L_{\infty}$ 范数)** | $$\lVert x \rVert_{\infty} = \max_{i=1}^{n} \|x_i\| = \max(\|x_1\|, \|x_2\|, \dots, \|x_n\|)$$ | 所有分量绝对值中的**最大值**   |

## 解的稳定性
![[零解的稳定性态.png|300]]
### **解的稳定性**的严格**李雅普诺夫 (Lyapunov) 定义**
####  零解 $x=0$ 的稳定性（李雅普诺夫意义下）
考虑一个微分方程组或动力系统，其平衡解为 $x=0$。假设该方程组为：

$$\frac{dx}{dt} = F(x, t)$$

如果零解 $x=0$ 是稳定的，则必须满足以下条件：

---

**定义：**
若对于**任意给定的正数 $\epsilon > 0$**（衡量解能偏离平衡点 $x=0$ 的最大程度），
**存在**一个正数 $\delta = \delta(\epsilon, t_0) > 0$（通常取决于 $\epsilon$ 和初始时间 $t_0$），
使得对于**任意初始值 $x_0$**，只要它满足：
$$\lVert x_0 \rVert \le \delta$$
则由初始条件 $x(t_0) = x_0$ 所确定的解 $x(t)$ 在**一切时间 $t \ge t_0$** 上都有：
$$\lVert x(t) \rVert \le \epsilon$$
则称零解 $x=0$ 是**稳定**的 (Stable)。

---

#### 与渐近稳定性的关系
如前所述，李雅普诺夫稳定性（简称李氏稳定）只保证了**解在平衡点附近保持有界**。它并不要求解必须返回平衡点。
如果要在李氏稳定性的基础上，要求解最终趋于零，则需要更强的条件，即**渐近稳定性 (Asymptotic Stability)**。
##### 渐近稳定性 (Asymptotic Stability)
如果零解 $x=0$ 满足以下两个条件，则它是渐近稳定的：
1. **李氏稳定：** 零解 $x=0$ 是稳定的（满足上述 $\epsilon-\delta$ 定义）。
2. 吸引性 (Attractivity)： 存在一个 $r > 0$，使得当 $\lVert x_0 \rVert \le r$ 时，解 $x(t)$ 最终趋于零，即：$$\lim_{t \to \infty} \lVert x(t) \rVert = 0$$
### 常系数线性微分方程组的稳定性
考虑一个常系数齐次线性微分方程组：
$$\frac{dx}{dt} = Ax$$
其中 $x$ 是 $n$ 维状态向量，而 $A$ 是一个 $n \times n$ 的常系数矩阵。系统的零解 $x(t) = 0$ 的稳定性完全取决于矩阵 $A$ 的所有特征值 $\lambda_i$ 的分布。
#### 1. 稳定性判据（基于特征值）
稳定性判据总结如下：

| **特征值 λi​ 的条件**                                                                                                                         | **零解 x=0 的稳定性**                  | **描述**                                    |
| --------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------- | ----------------------------------------- |
| **所有 $\text{Re}(\lambda_i) < 0$**                                                                                                       | **渐近稳定 (Asymptotically Stable)** | 所有解 $x(t)$ 随着时间 $t \to \infty$ 趋近于零。      |
| **所有 $\text{Re}(\lambda_i) \le 0$ 且**：<br>a) $\text{Re}(\lambda_i)=0$ 对应的特征值是**单根**，或<br>b) 如果是重根，其**几何重数等于代数重数** (即矩阵 $A$ 是**可对角化**的)。 | **李雅普诺夫稳定 (Stable)**             | 解 $x(t)$ 保持有界，但不一定趋于零。                    |
| **存在至少一个 $\text{Re}(\lambda_i) > 0$**                                                                                                   | **不稳定 (Unstable)**               | 存在解 $x(t)$ 随着时间 $t \to \infty$ 发散（趋向无穷大）。 |
| **存在 $\text{Re}(\lambda_i) = 0$ 且**：<br>对应的特征值是**重根**，但**几何重数小于代数重数**（即矩阵 $A$ 包含**约旦块** $J_k, k>1$）。                                    | **不稳定 (Unstable)**               | 解中会出现 $t^m e^{i\omega t}$ 形式的项，导致解随时间发散。  |

---
### 2. 特征值与解的形式
解 $x(t)$ 的基本形式由 $e^{\lambda t}$ 项构成。特征值 $\lambda = \alpha + i\beta$ 决定了系统解的动态行为：
- **实部 $\text{Re}(\lambda) = \alpha$：** 决定了振幅的变化趋势。
    - $\alpha < 0$: 振幅**衰减**（趋于稳定）。
    - $\alpha > 0$: 振幅**增长**（不稳定）。
    - $\alpha = 0$: 振幅**不变**（保持稳定或不稳定）。
        
- **虚部 $\text{Im}(\lambda) = \beta$：** 决定了解是否振荡。
    
    - $\beta \ne 0$: 解包含 $\cos(\beta t)$ 和 $\sin(\beta t)$ 项，呈**振荡**行为。
        
    - $\beta = 0$: 解是纯指数形式 $e^{\alpha t}$，无振荡。
        

### 总结要点

1. **渐近稳定**是最好的结果：所有特征值的实部都必须严格小于零。
    
2. **纯虚特征值 $(\text{Re}(\lambda)=0, \beta \ne 0)$**：
    
    - 如果是**单根**，系统是**稳定**的（解是常幅度的振荡）。
        
    - 如果是**不可对角化**的**重根**，系统是**不稳定**的（解会随时间线性增长）。

### 线性化稳定性定理 (Linearization Stability Theorem)

考虑一个非线性自治微分方程组：

$$\frac{dx}{dt} = F(x)$$

其中 $x$ 是状态向量，且 $x^*$ 是方程的一个平衡点（即 $F(x^*) = 0$）。

#### 1. 线性化过程

首先，在平衡点 $x^*$ 附近对非线性函数 $F(x)$ 进行泰勒级数展开，并保留一阶项。这相当于引入一个新变量 $y = x - x^*$，并将系统近似为：

$$\frac{dy}{dt} \approx A y$$

其中 $A$ 是**雅可比矩阵 (Jacobian Matrix)** 在平衡点 $x^*$ 处的取值：

$$A = \left. \frac{\partial F}{\partial x} \right|_{x=x^*}$$

#### 2. 定理内容（基于雅可比矩阵 $A$ 的特征值 $\lambda_i$）

线性化稳定性定理指出，非线性系统 $\frac{dx}{dt} = F(x)$ 在平衡点 $x^*$ 附近的稳定性，与线性化系统 $\frac{dy}{dt} = Ay$ 的零解 $y=0$ 的稳定性之间存在以下关系：

|**线性化系统稳定性**|**非线性系统稳定性（x∗ 的稳定性）**|**结论**|
|---|---|---|
|**渐近稳定**|**渐近稳定**|**所有**特征值 $\text{Re}(\lambda_i) < 0$。线性系统的结论完全适用于非线性系统。|
|**不稳定**|**不稳定**|**至少存在一个**特征值 $\text{Re}(\lambda_i) > 0$。线性系统的结论完全适用于非线性系统。|
|**李氏稳定**|**可能稳定、可能不稳定、可能渐近稳定**|**所有**特征值 $\text{Re}(\lambda_i) \le 0$，且**存在** $\text{Re}(\lambda_i) = 0$ 的特征值。**线性化失效。** 必须使用李雅普诺夫第二方法等其他非线性方法进行分析。|

---

### 核心意义
- **有效性 (Validity):** 如果线性化系统的特征值**实部全部非零**（即没有 $\text{Re}(\lambda_i) = 0$ 的特征值），则称该平衡点为**双曲平衡点 (Hyperbolic Equilibrium Point)**。对于双曲平衡点，线性化稳定性定理**完全有效**。
- **失效性 (Failure):** 如果线性化系统存在一个或多个纯虚特征值（即 $\text{Re}(\lambda_i) = 0$），则线性化方法**失效**。在这种临界情况下，非线性项的影响变得与线性项同等重要，需要借助更复杂的非线性分析工具，例如李雅普诺夫第二方法。
##  **李雅普诺夫函数方法概述**
李雅普诺夫方法由俄国数学家亚历山大·李雅普诺夫（Aleksandr Lyapunov）在 19 世纪末提出，主要用于**无需求解微分方程**的情况下，判断系统解的稳定性。它将系统的稳定性与一个特殊的标量函数（即李雅普诺夫函数）的性质联系起来。
该方法分为两类：
1. **李雅普诺夫第一方法 (First Method)：** 通过线性化系统（雅可比矩阵）来判断平衡点附近的局部稳定性。
2. **李雅普诺夫第二方法 (Second Method 或 Direct Method)：** 通过构造一个能量状的标量函数（李雅普诺夫函数）来判断稳定性和渐近稳定性，这是最常用和最强大的方法。

---

### **李雅普诺夫第二方法的核心**

假设我们有一个自治系统：

$$\dot{\mathbf{x}} = \mathbf{f}(\mathbf{x})$$

其中 $\mathbf{x} = \mathbf{0}$ 是一个平衡点（即 $\mathbf{f}(\mathbf{0}) = \mathbf{0}$）。

#### 1. **李雅普诺夫函数 $V(\mathbf{x})$**

李雅普诺夫函数 $V(\mathbf{x})$ 是一个标量函数，它必须满足以下两个基本条件：

- **正定性 (Positive Definite)：** 在平衡点的邻域内，
    
    - $V(\mathbf{0}) = 0$
        
    - $V(\mathbf{x}) > 0$（当 $\mathbf{x} \ne \mathbf{0}$ 时）
        

可以把它想象成系统的一种**“能量”**或**“距离”**度量，在平衡点处能量最小（为零）。

#### 2. **沿系统轨迹的导数 $\dot{V}(\mathbf{x})$**

$\dot{V}(\mathbf{x})$ 是 $V(\mathbf{x})$ 沿系统轨迹的全导数，根据链式法则计算：

$$\dot{V}(\mathbf{x}) = \frac{\partial V}{\partial \mathbf{x}} \cdot \dot{\mathbf{x}} = \nabla V(\mathbf{x}) \cdot \mathbf{f}(\mathbf{x})$$

$\dot{V}(\mathbf{x})$ 表示系统的“能量”随时间的变化率。

### **稳定性判据**

根据 $\dot{V}(\mathbf{x})$ 的性质，我们可以得出关于平衡点 $\mathbf{x}=\mathbf{0}$ 的稳定性结论：

|**V˙(x) 的性质**|**稳定性结论**|**物理含义**|
|---|---|---|
|**负半定 (Negative Semi-definite)**: $\dot{V}(\mathbf{x}) \le 0$|**李雅普诺夫稳定 (Stable)**|系统的“能量”不增加。|
|**负定 (Negative Definite)**: $\dot{V}(\mathbf{x}) < 0$ (当 $\mathbf{x} \ne \mathbf{0}$ 时)|**渐近稳定 (Asymptotically Stable)**|系统的“能量”持续减少，轨迹最终收敛到平衡点。|
|**正定 (Positive Definite)**: $\dot{V}(\mathbf{x}) > 0$|**不稳定 (Unstable)**|系统的“能量”持续增加，轨迹远离平衡点。|

#### **拉萨尔不变集原理 (LaSalle's Invariance Principle)**

如果 $\dot{V}(\mathbf{x})$ 只是负半定（$\dot{V}(\mathbf{x}) \le 0$），但 $\dot{V}(\mathbf{x})$ 不恒为零，拉萨尔原理可以提供一个更强的结论：系统会渐近收敛到 $\dot{V}(\mathbf{x}) = 0$ 的集合（称为**不变集**）上。

###  **李雅普诺夫方法的挑战**

李雅普诺夫方法的主要困难在于：

- **函数构造：** 目前没有通用的、系统性的方法来构造一个满足所有条件的 $V(\mathbf{x})$ 函数。
    
- **尝试与错误：** 找到一个合适的 $V(\mathbf{x})$ 往往需要经验、对系统物理意义的理解（例如，使用总能量作为 $V$）以及一些系统性的方法（如 **二次型函数**）。


## 平面自治系统定性分析

###  稳定性：相平面与轨线**

#### **一、基本方程与概念**

1. **平面一阶常微分方程组 (6.15)**
  $$\begin{cases} \frac{dx}{dt} = X(x, y), \\ \frac{dy}{dt} = Y(x, y), \end{cases} $$

    其中 $X, Y$ 对 $(x, y)$ 具有连续偏导数 。
2. **几何解释** 
    - **积分曲线**：解 $(x(t), y(t))$ 在三维空间 $Otxy$ 中形成的一条曲线 4。
    - **相平面**：平面 $Oxy$ 称为相平面 5。
    - **轨线 (轨道)**：积分曲线在相平面 $Oxy$ 上的投影称为轨线 6。
3. 转化为一阶常微分方程 
    若 $X^2 + Y^2$ 不恒为零，方程组可化为描述轨线形状的一阶方程：$$\frac{dy}{dx} = \frac{Y(x, y)}{X(x, y)} \quad (X(x, y) \neq 0), \quad \text{或} \quad \frac{dx}{dy} = \frac{X(x, y)}{Y(x, y)} \quad (Y(x, y) \neq 0).$$
    - **轨线不相交性**：由于满足解的存在唯一性条件，相平面上方程组 (6.15) 的轨线不能相交 

---

###  奇点
#### **二、奇点的定义与线性近似**
1. **奇点 (平衡点)** 
    - 定义：满足 $X(x^*, y^*) = 0$ 和 $Y(x^*, y^*) = 0$ 的常数解 $(x^*, y^*)$ 对应的点，称为方程组 (6.15) 的**奇点**（或**平衡点**、**平衡态**） 。
    - 奇点是系统**静止不动**的状态。
2. 线性近似方程组 
    通过平移将奇点移至原点，并取其线性项，得到线性近似方程组：$$\begin{cases} \frac{dx}{dt} = ax + by, \\ \frac{dy}{dt} = cx + dy, \end{cases} \quad A=\begin{pmatrix}a&b\\ c&d\end{pmatrix} $$
    - **唯一奇点假设**：假设 $\det A = \left|\begin{smallmatrix}a&b\\ c&d\end{smallmatrix}\right| \ne 0$ ，则原点 $(0, 0)$ 是方程组在相平面上的唯一奇点 。
#### **三、奇点类型分类（线性系统）**
奇点类型由系数矩阵 $A$ 的**特征方程**的根 $\lambda$ 决定 。
**特征方程**：$\lambda^{2}+p\lambda+q=0$ ，其中 $p = -(a+d)$（迹的相反数），$q = ad-bc$（行列式），$\Delta = p^{2}-4q$（判别式） 

| **情形 (按特征根 λ)**                                            | **判别条件**              | **奇点类型**              | **稳定性**                                                               | **相图特征**                           |
| ---------------------------------------------------------- | --------------------- | --------------------- | --------------------------------------------------------------------- | ---------------------------------- |
| I. 同号相异实根 $\lambda_1 \ne \lambda_2$                        | $q > 0, \Delta > 0$   | **结点 (Node)**         | $\lambda_1, \lambda_2 < 0$ (稳定结点)； $\lambda_1, \lambda_2 > 0$ (不稳定结点) | 除两条特征方向的轨线外，其余轨线均切于原点 。            |
| II. 异号实根 $\lambda_1 \lambda_2 < 0$                         | $q < 0$               | **鞍点 (Saddle Point)** | **不稳定** (零解不稳定)                                                       | 仅沿两条特征方向的轨线趋于或离开原点；其他轨线均穿过附近 。     |
| III. 重根 $\lambda_1 = \lambda_2 = \lambda$                  | $\Delta = 0$          | **退化结点** 或 **奇结点**    | $\lambda < 0$ (稳定)； $\lambda > 0$ (不稳定)                               | 退化结点：轨线切于一个方向。奇结点：所有轨线均为通过原点的半射线 。 |
| IV. 非零实部的复根 $\lambda = \alpha \pm i\beta$ ($\alpha \ne 0$) | $\Delta < 0, p \ne 0$ | **焦点 (Focus)**        | $\alpha < 0$ (稳定焦点)； $\alpha > 0$ (不稳定焦点)                             | 轨线为对数螺旋线，向内或向外盘旋 。                 |
| V. 纯虚根 $\lambda = \pm i\beta$                              | $\Delta < 0, p = 0$   | **中心 (Center)**       | **稳定但非渐近稳定**                                                          | 轨线为以原点为中心的同心圆（或椭圆）。                |

- **线性奇点定理 (Theorem 6)** ：在线性近似满足 $\det A \ne 0$ 的条件下，奇点的类型完全由特征方程的根决定

### **§ 6.2.2 极限环**

#### **四、极限环的概念与稳定性**
1. **极限环**
    - 定义：相平面上**孤立**的**闭轨线**（周期解）。
    - **孤立**意味着在它的邻域内没有其他的闭轨线。
    - 它代表了系统在相平面上长时间运行的**周期性震荡**。
2. **极限环的稳定性** 
    - **稳定极限环 (吸引型)**：邻域内轨线均向其靠拢 
    - **不稳定极限环 (排斥型)**：邻域内轨线均远离 
    - **半稳定极限环**：一侧轨线靠拢，另一侧轨线远离 
#### **五、判断极限环存在的定理**

1. **环域定理 (Poincaré–Bendixson Theorem 的推广)** 
    
    - 设 $D$ 为一个有界的**环形闭域**，且 $D$ 内不含奇点 。

    - 若经过 $D$ 上任一点的轨线始终不离开 $D$ 。

    - **结论**：则该轨线本身是一个周期解，或最终趋于 $D$ 内的某个周期解（闭轨线）。
        
2. **Bendixson 判别准则 (Bendixson Criterion) 
    
    - 若在单连通区域 $D^*$ 中 $\frac{\partial X}{\partial x} + \frac{\partial Y}{\partial y}$（即散度 $\text{div} \mathbf{F}$）**不变号**（始终 $\ge 0$ 或 $\le 0$），且在 $D^*$ 任何子域内不恒为零 。
        
    - **结论**：则系统在 $D^*$ 内**不存在**周期解，也**不存在**极限环 。
        
    - _直观理解_：散度若保持单调，相流无法在区域内“循环”。
