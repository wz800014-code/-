# 第一章 概率论的基础知识

### §1 随机试验 
   **定义:** 对随机现象的观察，称为**随机试验**（简称试验），记为 $E$。
* **特点 :** 
    1.  可在相同条件下重复进行。
    2.  试验结果可能不止一个，但能明确所有的可能结果 。
    3.  试验前无法确定是哪个结果会出现 。
### §2 样本空间、随机事件 
**样本空间 ($\Omega$):** 试验的所有可能结果所组成的集合，记为 $\Omega=\{\omega\}$ 。
**样本点 ($\omega$):** 试验的每一个结果或样本空间的每个元素。
**随机事件 (事件):** 试验中可能出现也可能不出现的情况，记作 $A, B, C$ 等 。任何事件均对应着样本空间的某个**子集**。
* **特殊事件 :**
    * **必然事件:** 样本空间 $\Omega$ 。
    * **不可能事件:** 空集 $\emptyset$。
    * **基本事件:** 只包含一个样本点的事件 $\{\omega\}$ 。
* **事件间的关系:** 
*1.**包含关系:** $A \subset B$（$A$ 发生必导致 $B$ 发生。)
2.**和事件**：事件A与B至少有一个发生$\Leftrightarrow$A$\bigcup$B发生
3.**积事件**：A与B同时发生$\Leftrightarrow$A$\bigcap$B=AB发生
4.**差事件：**A-B发生$\Leftrightarrow$事件A发生而事件B不发生
5.**互斥的事件：**AB=$\emptyset$
6.**互逆的事件：**若A$\bigcup$B＝ Ω, 且AB＝$\emptyset$，称A与B互为逆事件. 表示A,B不可能同时发生，但必有一个发生。
#### 事件的运算
 ##### 1. 交换律 (Commutative Law)
- $A \cup B = B \cup A$
- $AB = BA$（注：$AB$ 通常表示交集 $A \cap B$）
##### 2. 结合律 (Associative Law)
- $(A \cup B) \cup C = A \cup (B \cup C)$
- $(AB)C = A(BC)$
##### 3. 分配律 (Distributive Law)
- $(A \cup B)C = (AC) \cup (BC)$
- $(AB) \cup C = (A \cup C)(B \cup C)$
---
##### 4. 对偶律 / 德·摩根定律 (De Morgan's Laws)
这是集合运算中非常重要的变换规则，描述了“补集”与“并、交运算”的关系：
- **基本形式：**
    - $\overline{A \cup B} = \bar{A} \cap \bar{B}$ （并的补等于补的交）
    - $\overline{AB} = \bar{A} \cup \bar{B}$ （交的补等于补的并）
- **可推广形式 (Generalization)：**
    - $\overline{\bigcup\limits_{k} A_k} = \bigcap\limits_{k} \bar{A}_k$
    - $\overline{\bigcap\limits_{k} A_k} = \bigcup\limits_{k} \bar{A}_k$
* **例1 (掷骰子):** $E_{4}$: 掷一颗骰子，考察可能出现的点数。
    * 样本空间: $S_{4}=\{1,2,3,4,5,6\}$ 。
    * 事件 $A=$“掷出偶数点” $=\{2,4,6\}$。
    * 事件 $B=$“掷出大于4的点” 。
#### §3 古典概型与概率 
* **概率的性质:**
    * (1) $P(\emptyset)=0$。
    * (2) **有限可加性:** 设 $A_{1}, \ldots, A_{n}$ 是 $n$ 个两两互不相容的事件 ($A_{i} A_{j}=\emptyset$), 则$$P(A_{1} \cup A_{2} \cup \ldots \cup A_{n}) = P(A_{1}) + P(A_{2}) + \ldots + P(A_{n})$$
    * (4) **事件差:** $P(A-B) = P(A) - P(AB)$。
    * (5) **加法公式:** 对任意两事件 $A, B$，有$$P(A \cup B) = P(A) + P(B) - P(AB)$$
    * (6)**互补性:** $P(\bar{A}) = 1 - P(A)$ 。

* **例 (古典概型抽球公式):** 盒中有 $N$ 个球，其中 $M$ 个白球，现从中任抽 $n$ 个球，则这 $n$ 个球中恰有 $k$ 个白球的概率是:$$P = \frac{C_{M}^{k} C_{N-M}^{n-k}}{C_{N}^{n}}$$
#### §4 条件概率、独立性
* **条件概率 (定义):** 设 $A, B$ 为两个事件，且 $P(A) > 0$，则称 $B$ 在 $A$ 发生的条件下发生的概率为:
    $$P(B|A) = \frac{P(AB)}{P(A)}$$
* **乘法公式 (推导):** $P(AB) = P(A) P(B|A)$
* **两个事件的独立性 (定义 p14):** 若 $P(AB) = P(A) P(B)$，则称事件 $A$ 与 $B$ 相互独立。
* **例 (独立性证明):** 证明若 $A, B$ 相互独立, 则 $A, \bar{B}$ 相互独立。
    * **证明:**$$P(A \bar{B}) = P(A - AB) = P(A) - P(AB)$$ (事件差公式)
        * 因 $A, B$ 独立: $P(AB) = P(A)P(B)$ 。$$P(A \bar{B}) = P(A) - P(A)P(B) = P(A)[1 - P(B)] = P(A)P(\bar{B})$$
        * 故 $A$ 与 $\bar{B}$ 相互独立。
## §5 全概率公式与贝叶斯公式
* **全概率公式 (应用例题):**
    * **例1 (袋子抽球):** 甲袋（2白，1红），乙袋（2红，1白）。从甲袋中任取一球放入乙袋，再从乙袋任取一球是红球 ($B$) 的概率。
    * 设 $A_{1}$ - 从甲袋放入乙袋的是白球 ($P(A_{1}) = 2/3$)；$A_{2}$ - 从甲袋放入乙袋的是红球 ($P(A_{2}) = 1/3$)。
    * 若 $A_{1}$ 发生 (乙袋 2白2红), $P(B|A_{1})=2/4=1/2$。
    * 若 $A_{2}$ 发生 (乙袋 1白3红), $P(B|A_{2})=3/4$。
    * **全概率公式:** $P(B) = P(B|A_{1}) P(A_{1}) + P(B|A_{2}) P(A_{2})$$$P(B) = \frac{2}{4} \cdot \frac{2}{3} + \frac{3}{4} \cdot \frac{1}{3} = \frac{4}{12} + \frac{3}{12} = \frac{7}{12}$$
* **贝叶斯公式**
	* $$P(A_j|B)=\frac{P(A_j)P(B|A_j)}{\sum^{n}_{i=1}{P(A_i)P(B|A_i)}},(j=1,\dots,n)$$

# 第二章 随机变量及其分布

## §1 随机变量

* **定义 :** 设 $S=\{e\}$ 是试验的样本空间, 如果量 $X$ 是定义在 $S$ 上的一个单值实值函数，则称 $X$ 为**随机变量** (r. v. $X$)
    * **映射:** $X: S \to R$ 
* **引入意义:** 将随机试验的结果数量化，描述随机事件 。
* **分类:** 离散型随机变量、非离散型（连续型、奇异型/混合型）。
## §2 离散型随机变量的分布律
* **定义:** 若 $X$ 取值 $x_{1}, x_{2}, \ldots, x_{k}, \ldots$ 且取这些值的概率依次为 $p_{1}, p_{2}, \ldots, p_{k}, \ldots$，则称$$P\{X=x_{k}\} = p_{k} \quad (k=1, 2, \ldots)$$为 $X$ 的**分布律**。
* **性质**：
	* 1.$p_k\geq0 ,k=1,2,3,\dots$;
	* 2.$\sum_{k\geq1}p_k=1$.
### 常见的离散型随机变量的分布律
#### 1.两点分布

| $X$   | 1     | 0   |
| ----- | ----- | --- |
| $p_k$ | $1-p$ | $p$ |
#### 2.二项分布
记作$X\sim b(n,p)$
其中：
	$X$表示n重贝努里试验中事件$A$发生的次数
	$n$表示伯努利试验发生的次数
	$p=p(A)$
$$P\{X = k\} = C_n^k p^k (1 - p)^{n - k},(k=0,1,2,3,\dots,n)$$
#### 3.泊松分布(n很大，p很小的二项分布)
如果一个随机变量 $X$ 服从参数为 $\lambda$ 的泊松分布即$X\sim \pi{(\lambda)}$，其概率质量函数为：
$$P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!}, \quad (k = 0, 1, 2, \dots)$$
- **$\lambda$ (Lambda)**：这是唯一的参数，代表事件发生的**平均速率**（或平均发生次数）。
- **$e$**：自然常数（约等于 2.718）。
- **$k$**：你想要计算的实际发生次数。
* **例 ** 问讯次数 $X \sim P(\lambda)$。已知 $P\{X=0\} = e^{-6}$
    * **求参数 $\lambda$:** $P\{X=0\} = \frac{\lambda^{0}}{0!} e^{-\lambda} = e^{-\lambda}$。故 $\lambda = 6$。
    * **求 $P\{X \ge 2\}$:**
        * **泊松分布公式:** $P\{X=k\} = \frac{\lambda^{k}}{k!} e^{-\lambda}$$$P\{X \ge 2\} = 1 - P\{X < 2\} = 1 - (P\{X=0\} + P\{X=1\})$$$$P\{X \ge 2\} = 1 - (e^{-6} + \frac{6^{1}}{1!} e^{-6}) = 1 - 7 e^{-6} \approx 0.9826$$

## §3 随机变量的分布函数 
* **定义:** 设 $X$ 是一个随机变量，**分布函数** $F(x)$ 定义为:$$F(x) = P\{X \le x\}, \quad -\infty < x < +\infty$$
* $$P\{a<X\le b\}=P\{X\le b\}-P\{X\le a\}=F(b)-F(a)$$
* **性质：**
	* 1.单调不减性 (Monotonicity)：若 $x_1 < x_2$，则 $F(x_1) \le F(x_2)$
	* 2.归一性 (Normalization / Boundedness)：$0 \le F(x) \le 1$，且 $F(-\infty) = 0, F(+\infty) = 1$
	* 3.右连续性 (Right-continuity)：$\lim_{x \to x_0^+} F(x) = F(x_0)$
* **一些公式：**
	* $P(a < X \le b) = F(b) - F(a)$
	* $P(X > a) = 1 - F(a)$
	* $P(X = a) = F(a) - F(a-0)$
	* $P(X < a) = F(a-0)$
## §4 连续型随机变量及其概率密度
* **概率密度函数 ($f(x)$) (定义):** 若 $X$ 的分布函数为 $F(x)$ 存在非负函数 $f(x)$, 使得 $F(x) = \int_{-\infty}^{x} f(t) dt$, 则称 $f(x)$ 为 $X$ 的**概率密度函数**。
* **性质:**
    1.  $f(x) \ge 0$
    2.  归一性: $\int_{-\infty}^{\infty} f(x) dx = 1$ 
* **概率计算:** $P\{a < X < b\} = \int_{a}^{b} f(x) dx = F(b) - F(a)$

### 几个常用的连续型分布
#### 1.均匀分布
如果随机变量 $X$ 在区间 $[a, b]$ 上服从均匀分布，记作 $X \sim U(a, b)$。它的概率密度函数是一条水平直线：

$$f(x) = \begin{cases} \frac{1}{b - a}, & a \le x \le b \\ 0, & \text{其他} \end{cases}$$
$$F(x) = P(X \le x) = \begin{cases} 0, & x < a \\ \frac{x - a}{b - a}, & a \le x < b \\ 1, & x \ge b \end{cases}$$
#### 2.指数分布
如果随机变量 $X$ 服从参数为 $\lambda$ 的指数分布（其中 $\lambda > 0$），记作 $X \sim E(\lambda)$。它的概率密度函数为：$$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x \ge 0 \\ 0, & x < 0 \end{cases}$$$$F(x) = P(X \le x) = \begin{cases} 1 - e^{-\lambda x}, & x \ge 0 \\ 0, & x < 0 \end{cases}$$
#### 3.正态分布
如果随机变量 $X$ 服从均值为 $\mu$、方差为 $\sigma^2$ 的正态分布，记作 $X \sim N(\mu, \sigma^2)$。其密度函数公式为：$$f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$
当 $\mu = 0, \sigma = 1$ 时，称为标准正态分布，记为 $Z \sim N(0, 1)$。
  $$\varphi(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^{2}}{2}}, \quad -\infty < x < +\infty$$ $$\Phi(x) = P\{X \le x\} = \int_{-\infty}^{x} \varphi(t) dt$$
其分布函数通常用专用符号 $\Phi(x)$ 表示。任何一般的正态分布都可以通过以下公式标准化：
$$Z = \frac{X - \mu}{\sigma}$$

* **例1 (求参数 A, 概率, 分布函数):** 已知 $f(x)=\begin{cases}A x^{2} & 0 < x < 1 \\ 0 & \text{其他}\end{cases}$
    * **(1) 求参数 A (推导):** 由 $\int_{0}^{1} A x^{2} dx = 1 \Rightarrow A \cdot \frac{x^{3}}{3} |_{0}^{1} = \frac{A}{3} = 1 \Rightarrow A = 3$。
    * **(2) 求 $P\{0.5 < X < 3\}$ (推导):** $P\{0.5 < X < 3\} = \int_{0.5}^{1} 3 x^{2} dx = x^{3} |_{0.5}^{1} = 1 - (0.5)^{3} = 0.875$。
    * **(3) 求分布函数 $F(x)$ (推导):**$$F(x)=\begin{cases}0 & x \le 0 \\ \int_{0}^{x} 3 t^{2} dt = x^{3} & 0 < x < 1 \\ 1 & x \ge 1\end{cases}$$

##### **无记忆性**
如果随机变量 $X$ 具有无记忆性，对于任何正数 $s$ 和 $t$，以下等式成立：
$$P(X > s + t \mid X > s) = P(X > t)$$

## §5 随机变量函数的分布

* **公式法:** 若 $X \sim f_{X}(x)$, $Y=g(x)$ 是严格单调可导函数, 则 $Y$ 的概率密度函数 $f_{Y}(y)$ 为:$$f_{Y}(y) = f_{X}[g^{-1}(y)] \cdot \left| [cite_start]\frac{d g^{-1}(y)}{d y} \right|$$
* **例4 (线性变换 - 推导):** 已知 $X \sim N(\mu, \sigma^{2})$, 求 $Y = \frac{X-\mu}{\sigma}$ 的概率密度。
    * **反函数:** $x = g^{-1}(y) = \sigma y + \mu$(p54)。
    * **导数:** $\left| \frac{d g^{-1}(y)}{d y} \right| [cite_start]= \sigma$  (p54)。
    * **结果:** $f_{Y}(y) = f_{X}(\sigma y + \mu) \cdot \sigma = \frac{1}{\sqrt{2\pi}} e^{-\frac{y^{2}}{2}}$。
    * **结论:** $Y \sim N(0, 1)$ (p54)。

# 第三章 多维随机变量及其分布
## §1 二维随机变量的分布 
* **联合分布函数 :** 设 $(X, Y)$ 是二维随机变量, **联合分布函数** $F(x, y)$ 定义为:$$F(x, y) = P\{X \le x, Y \le y\}$$
* **矩形区域概率公式 (推导):** 对于 $x_{1} < x_{2}, y_{1} < y_{2}$, 矩形区域概率为:$$P\{x_{1} < X \le x_{2}, y_{1} < Y \le y_{2}\} = F(x_{2}, y_{2}) - F(x_{1}, y_{2}) - F(x_{2}, y_{1}) + F(x_{1}, y_{1})$$
* **性质：**
	* 1.归一性：
		* $$F(+\infty, +\infty) = \lim_{\substack{x \to +\infty \\ y \to +\infty}} F(x, y) = 1$$
		* $$F(-\infty, -\infty) = \lim_{\substack{x \to -\infty \\ y \to -\infty}} F(x, y) = 0$$
		* $$F(-\infty, y) = \lim_{\substack{x \to -\infty }} F(x, y) = 0$$
		* $$F( x,-\infty) = \lim_{\substack{y \to -\infty }} F(x, y) = 0$$
	* 单调不减性
	* 右连续性
	* **矩形不等式：****随机点 $(X, Y)$ 落在矩形区域 $x_1 < X \le x_2, y_1 < Y \le y_2$ 内的概率必须大于或等于 0。**
		对于任意满足 $x_1 < x_2$ 且 $y_1 < y_2$ 的两点，有：$$F(x_2, y_2) - F(x_1, y_2) - F(x_2, y_1) + F(x_1, y_1) \ge 0$$
### 二维离散型随机变量
如果二维随机变量 $(X, Y)$ 所有可能取到的值是**有限对**或者**可列无限多对**，我们就称它为二维离散型随机变量。
###  联合概率分布律 
$$P\{X = x_i, Y = y_j\} = p_{ij} \quad (i, j = 1, 2, \dots)$$
* 性质：
	- **非负性：** $p_{ij} \ge 0$。
	- **归一性：** $\sum_i \sum_j p_{ij} = 1$（所有可能取值的概率之和为 1）。

通常我们用**分布表格**来表示：

| $X$∖$Y$     | $y_1$​   | $y_2$​   | **…**   | $y_j​$   | **…**   |
| ----------- | -------- | -------- | ------- | -------- | ------- |
| **$x_1$**   | $p_{11}$ | $p_{12}$ | $\dots$ | $p_{1j}$ | $\dots$ |
| **$x_2$**   | $p_{21}$ | $p_{22}$ | $\dots$ | $p_{2j}$ | $\dots$ |
| **$\dots$** | $\dots$  | $\dots$  | $\dots$ | $\dots$  | $\dots$ |
### 二维连续型随机变量
* **联合概率密度函数 ($f(x, y)$) (定义):** 若存在非负函数 $f(x, y)$ 使得 $F(x, y) = \int_{-\infty}^{y} \int_{-\infty}^{x} f(u, v) du dv$, 则称 $f(x, y)$ 为联合概率密度函数。
    * **性质 (归一性):** $$\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} f(x, y) dx dy = 1$$。
    * **关系:** $$f(x, y) = \frac{\partial^{2} F(x, y)}{\partial x \partial y}$$
    * 对于任意平面区域$G\subset R^2$,$$P\{(X,Y)\in G\}=\iint_G f(x,y)dxdy$$
#### 两个常用的二维连续型分布
##### 1.二维均匀分布
设 $D$ 是平面上的一个有界区域，其面积为 $S_D$。如果随机变量 $(X, Y)$ 的联合概率密度函数 $f(x, y)$ 为：$$f(x, y) = \begin{cases} \frac{1}{S_D}, & (x, y) \in D \\ 0, & (x, y) \notin D \end{cases}$$
那么称 $(X, Y)$ 在 $D$ 上服从均匀分布。

##### 2.二维正态分布
一个二维随机变量 $(X, Y)$ 服从均值为 $(\mu_1, \mu_2)$，方差为 $(\sigma_1^2, \sigma_2^2)$，相关系数为 $\rho$ 的正态分布，记作 $(X, Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$。
其联合概率密度函数公式如下：
$$f(x, y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left[ -\frac{1}{2(1-\rho^2)} \left( \frac{(x-\mu_1)^2}{\sigma_1^2} - \frac{2\rho(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2} \right) \right]$$

其中：
- **$\mu_1, \mu_2$**: 分别是 $X$ 和 $Y$ 的数学期望，决定了分布中心的位置。
- **$\sigma_1, \sigma_2$**: 分别是 $X$ 和 $Y$ 的标准差，决定了分布在轴向上的“胖瘦”。
- **$\rho$**: 相关系数，反映了 $X$ 和 $Y$ 之间的线性相关程度（$-1 \le \rho \le 1$）。
###### 重要性质
- **边缘分布依然是正态的**：
    $X$ 独立服从 $N(\mu_1, \sigma_1^2)$，$Y$ 独立服从 $N(\mu_2, \sigma_2^2)$。但反之不一定成立（即边缘分布为正态，联合分布未必是正态）。

- **线性变换不变性**：
    $X$ 和 $Y$ 的线性组合（如 $aX + bY$）仍然服从一维正态分布。

- **独立性与相关性的等价性**：
    对于一般的随机变量，独立一定不相关，但不相关不一定独立。但在二维正态分布中，独立与不相关（$\rho = 0$）是互为充要条件的。

- **等高线是椭圆**：
    在 $x-y$ 平面上，概率密度相等的点轨迹是一个椭圆。椭圆的长短轴方向和倾斜度由 $\rho$ 和 $\sigma$ 决定。
## §2 边缘分布 
### 边缘分布函数
无论是离散还是连续，从联合累积分布函数 $F(x, y) = P(X \le x, Y \le y)$ 获取边缘分布函数的方法是相同的：**让另一个变量趋于无穷大。**
- $X$ 的边缘分布函数：$$F_X(x) = P(X \le x) = F(x, +\infty) = \lim_{y \to +\infty} F(x, y)$$
- $Y$ 的边缘分布函数：$$F_Y(y) = P(Y \le y) = F(+\infty, y) = \lim_{x \to +\infty} F(x, y)$$
### 边缘分布律
#### 1. 离散型随机变量
对于离散型随机变量 $(X, Y)$，其联合分布律为 $P(X=x_i, Y=y_j) = p_{ij}$。
- $X$ 的边缘分布律：是将 $Y$ 的所有可能取值求和：$$P(X=x_i) = \sum_{j=1}^{\infty} p_{ij}$$
- $Y$ 的边缘分布律：是将 $X$ 的所有可能取值求和：$$P(Y=y_j) = \sum_{i=1}^{\infty} p_{ij}$$
> **形象理解**：如果你把联合概率分布写成一个表格，$X$ 的边缘分布就是每一行的**行和**，$Y$ 的边缘分布就是每一列的**列和**。这也是为什么它被称为“边缘”的原因——它们写在表格的边框处。
---
#### 2. 连续型随机变量
对于连续型随机变量 $(X, Y)$，其联合概率密度函数为 $f(x, y)$。
- $X$ 的边缘概率密度函数：对 $y$ 在整个定义域上进行积分，把 $y$ “积掉”：$$f_X(x) = \int_{-\infty}^{\infty} f(x, y) \, dy$$
- $Y$ 的边缘概率密度函数：对 $x$ 在整个定义域上进行积分：$$f_Y(y) = \int_{-\infty}^{\infty} f(x, y) \, dx$$
### 边缘密度函数
#### 1. 数学定义
设二维连续型随机变量 $(X, Y)$ 的联合概率密度函数为 $f(x, y)$，则：
- $X$ 的边缘概率密度函数 $f_X(x)$：$$f_X(x) = \int_{-\infty}^{+\infty} f(x, y) \, dy$$含义：固定 $x$ 值，把所有可能的 $y$ 值发生的概率“累加”（积分）起来。
- $Y$ 的边缘概率密度函数 $f_Y(y)$：$$f_Y(y) = \int_{-\infty}^{+\infty} f(x, y) \, dx$$含义：固定 $y$ 值，把所有可能的 $x$ 值发生的概率“累加”（积分）起来。
---
#### 2. 几何理解
想象 $f(x, y)$ 是在 $x-y$ 平面上方起伏的一个三维曲面，曲面下方的总面积为 1。
- **求 $f_X(x)$ 的过程**：相当于沿着 $y$ 轴的方向对这个三维物体进行“挤压”或“投影”，把所有的质量都堆积到 $x$ 轴上。
- **物理意义**：如果你把这个分布想象成一团有质量的云雾，$f_X(x)$ 就是你从 $y$ 轴无穷远处观察这团云雾时，在 $x$ 轴上看到的投影密度。

## §3 随机变量的独立性

* **独立性 (定义):** $(X_{1}, \ldots, X_{n})$ 和 $(Y_{1}, \ldots, Y_{m})$ 相互独立，如果联合分布函数等于边缘分布函数的乘积:$$F(x_{1}, \ldots, x_{n}, y_{1}, \ldots, y_{m}) = F_{X}(x_{1}, \ldots, x_{n}) F_{Y}(y_{1}, \ldots, y_{m})$$
    * **二维连续型判定:** $X$ 与 $Y$ 相互独立 $\Leftrightarrow$ $f(x, y) = f_{X}(x) f_{Y}(y)$。
    * **二维离散型判定：**$P_{ij}=P\{X=x_i,Y=y_j\},i,j=1,2,\dots$,$X$ 与 $Y$ 相互独立 $\Leftrightarrow$ $P_{ij}= P_{i} P_{j}$。
* **定理 (二维正态分布):** 设 $(X, Y) \sim N(\mu_{1}, \mu_{2}, \sigma_{1}^{2}, \sigma_{2}^{2}, \rho)$, 则 $X$ 与 $Y$ 相互独立 $\Leftrightarrow$ **常数 $\rho=0$** 
## §4 两个随机变量函数的分布
### 一、二维离散型随机变量函数的分布律$P$
求解二维离散型随机变量函数的分布律，核心思想是将**函数值的取值情况**映射回**原始变量的联合分布**中。
假设二维离散型随机变量 $(X, Y)$ 的联合分布律为 $P(X=x_i, Y=y_j) = p_{ij}$，我们要讨论函数 $Z = g(X, Y)$ 的分布律。
#### 1. 求解步骤

1. 确定 $Z$ 的所有可能取值
    根据 $X$ 和 $Y$ 可能取值的组合 $(x_i, y_j)$，计算出所有对应的 $z_{k} = g(x_i, y_j)$。
2. 归并相同取值的概率
    对于 $Z$ 的每一个可能取值 $z_k$，寻找所有满足 $g(x_i, y_j) = z_k$ 的样本点 $(x_i, y_j)$。
3. 求和计算概率
    利用概率的可加性，计算 $P(Z=z_k)$：$$P(Z=z_k) = \sum_{g(x_i, y_j) = z_k} P(X=x_i, Y=y_j)$$
---
#### 2. 常见函数类型的分布
##### A. 和的分布：$Z = X + Y$
这是最常见的情况。如果 $X$ 和 $Y$ 取值为整数，则：
$$P(Z=z) = \sum_{i} P(X=x_i, Y=z-x_i)$$
> **注意：** 如果 $X$ 和 $Y$ 相互独立，则该公式演变为**离散卷积公式**：$P(Z=z) = \sum_{i} P(X=x_i) \cdot P(Y=z-x_i)$。
##### B. 最值分布：$M = \max(X, Y)$ 与 $N = \min(X, Y)$
- **最大值 $M$：** $M \le z$ 等价于 $X \le z$ 且 $Y \le z$。
- **最小值 $N$：** $N > z$ 等价于 $X > z$ 且 $Y > z$。
* **最大值 $M$ 和最小值 $N$ 的分布:** 设 $X_{1}, \ldots, X_{n}$ 相互独立, 分布函数分别为 $F_{1}(x_{1}), \ldots, F_{n}(x_{n})$ 。
    * **最大值 $M = \max\{X_{1}, \ldots, X_{n}\}$ 的分布函数:**$$F_{M}(z) = F_{1}(z) F_{2}(z) \ldots F_{n}(z)$$
    * **最小值 $N = \min\{X_{1}, \ldots, X_{n}\}$ 的分布函数:**$$F_{N}(z) = 1 - [1 - F_{1}(z)][1 - F_{2}(z)] \ldots [1 - F_{n}(z)]$$

### 二、多个连续型随机变量函数的密度函数$P$
#### 1. 分布函数法 (通用且最稳妥)
这是处理此类问题的基本思路。设 $Z = g(X, Y)$，步骤如下：
1. 先求累积分布函数 (CDF)：$$F_Z(z) = P(Z \le z) = P(g(X, Y) \le z) = \iint_{g(x,y) \le z} f(x, y) \, dx \, dy$$
这里的关键是确定积分区域 $D_z = \{ (x, y) \mid g(x, y) \le z \}$。
2. 求导得到概率密度 (PDF)：$$f_Z(z) = \frac{d}{dz} F_Z(z)$$
---
#### 2. 变量代换法 (雅可比矩阵法)
当你需要求从 $(X, Y)$ 到 $(U, V)$ 的联合分布变换时，此法非常高效。
假设有变换 $u = g_1(x, y)$ 和 $v = g_2(x, y)$，且其逆变换 $x = h_1(u, v), y = h_2(u, v)$ 存在且一、二阶偏导连续，则 $(U, V)$ 的联合密度函数为：$$f_{U,V}(u, v) = f_{X,Y}(h_1, h_2) \cdot |J|$$
其中 $J$ 为雅可比行列式（Jacobian）：$$J = \det \begin{pmatrix} \frac{\partial x}{\partial u} & \frac{\partial x}{\partial v} \\ \frac{\partial y}{\partial u} & \frac{\partial y}{\partial v} \end{pmatrix}$$
> **注意**：最后通常需要对 $v$ 求边缘分布来得到 $U$ 的密度：$f_U(u) = \int_{-\infty}^{+\infty} f_{U,V}(u, v) \, dv$。
---
#### 3. 常见特定函数的结论
##### A. 和的分布：$Z = X + Y$
如果 $X, Y$ 独立，其密度函数为两者的卷积：$$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x) \, dx$$
##### B. 商的分布：$Z = Y / X$
若 $X, Y$ 独立且 $X > 0$，则：$$f_Z(z) = \int_{0}^{+\infty} x f_X(x) f_Y(zx) \, dx$$

# 第四章 随机变量的数字特征
## §1 随机变量的数学期望 (均值)
* **离散型 r. v. 的期望:**
    * **定义:** $$E(X) = \sum_{k=1}^{\infty} x_{k} p_{k}$$ (要求 $\sum_{k=1}^{\infty} |x_{k}| p_{k} < \infty$) 

* **连续型 r. v. 的期望:**
    * **定义:** $$E(X) = \int_{-\infty}^{\infty} x f(x) dx$$ (要求 $\int_{-\infty}^{\infty} |x| f(x) dx < \infty$) 

* **函数期望:**
    * **一维:** $E[g(X)] = \int_{-\infty}^{\infty} g(x) f(x) dx$ (连续型)
    * **二维:** $E[g(X, Y)] = \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} g(x, y) f(x, y) dx dy$
### 期望的性质：
#### 1. 常数的期望
如果 $C$ 是一个常数，那么它的期望就是它本身。

$$E(C) = C$$
> **理解：** 常数没有波动，其平均值自然就是它自己。
#### 2. 线性性质（Linearity of Expectation）
这是期望最核心、最常用的性质，分为两部分：
- 倍数关系： 随机变量乘以常数 $a$，期望也扩大 $a$ 倍。$$E(aX) = aE(X)$$
- 和的期望： 两个随机变量之和的期望等于各自期望之和。$$E(X + Y) = E(X) + E(Y)$$
    注意： 这一条对于任意两个随机变量都成立，无论它们是否独立。
将上述结合，可以得到更通用的线性公式：$$E(aX + bY + c) = aE(X) + bE(Y) + c$$
#### 3. 独立随机变量的乘积
如果随机变量 $X$ 和 $Y$ 是相互独立的，那么它们乘积的期望等于各自期望的乘积：$$E(XY) = E(X)E(Y)$$
> **警示：** 如果 $X$ 和 $Y$ 不独立，这个等式通常不成立。
#### 4. 函数的期望（EUE定理）
若 $Y$ 是 $X$ 的函数，即 $Y = g(X)$，我们不需要求出 $Y$ 的分布，可以直接利用 $X$ 的概率密度或分布律求期望：
- **离散型：** $E[g(X)] = \sum_{i} g(x_i)P(X = x_i)$
- **连续型：** $E[g(X)] = \int_{-\infty}^{+\infty} g(x)f(x)dx$
### 5. 单调性
如果对于所有的取值，都有 $X \le Y$，那么它们的期望也满足：$$E(X) \le E(Y)$$
由此推论：如果 $X \ge 0$，则 $E(X) \ge 0$。
##### 常见随机变量的期望参考表

| **常用分布**   | **符号**             | **期望 E(X)**         |
| ---------- | ------------------ | ------------------- |
| **0-1 分布** | $B(1, p)$          | $p$                 |
| **二项分布**   | $B(n, p)$          | $np$                |
| **泊松分布**   | $P(\lambda)$       | $\lambda$           |
| **均匀分布**   | $U(a, b)$          | $\frac{a+b}{2}$     |
| **正态分布**   | $N(\mu, \sigma^2)$ | $\mu$               |
| **指数分布**   | $e(\lambda)$       | $\frac{1}{\lambda}$ |

## §2 方差 
* **定义:** $D(X) = E[(X - E(X))^{2}]$。
* **计算公式:** $D(X) = E(X^{2}) - [E(X)]^{2}$ 
* **性质 :**
    1.  $D(c)=0$ 若 $D(X)=0$，则 $P\{X=C\}=1$ 
    2.  **推导:** $D(aX + b) = a^{2} D(X)$ 
    3. 若X,Y独立，则$D(X+Y)=D(X)+D(Y)$
* **重要 r. v. 的方差:**
    1.  **二项分布 $b(n, p)$:** $D(X) = n p (1-p)$
    2.  **指数分布 $E(\lambda)$:** $D(X) = \frac{1}{\lambda^{2}}$ 
    3. **均匀分布：**$U(a,b)$:$D(X)=\frac{{b-a}^2}{12}$
    4. **泊松分布$P(\lambda)$:**$D(X)=\lambda$
### 切比雪夫不等式
**切比雪夫不等式（Chebyshev's Inequality）**是概率论中一个非常强大的工具，它描述了随机变量偏离其数学期望的可能性。
它的魅力在于：**无论随机变量服从什么分布**（只要方差存在），它都成立。
设随机变量 $X$ 具有数学期望 $E(X) = \mu$ 和方差 $D(X) = \sigma^2$。对于任意正数 $\epsilon$，有：$$P(|X - E(X)| \ge \epsilon) \le \frac{D(X)}{\epsilon^2}$$或者其等价形式（设 $\epsilon = k\sigma$，即 $k$ 倍标准差）：$$P(|X - \mu| \ge k\sigma) \le \frac{1}{k^2}$$
> **通俗理解：** 随机变量 $X$ 落在期望 $\mu$ 的 $k$ 倍标准差范围之外的概率，不会超过 $1/k^2$。

## §3 协方差与相关系数 
* **协方差 (定义):** $Cov(X, Y) = E[(X - E(X))(Y - E(Y))]$。
* **计算公式:** $Cov(X, Y) = E(XY) - E(X)E(Y)$。
* **不相关:** 若 $Cov(X, Y) = 0$，则称 $X$ 与 $Y$ 不相关
    * **性质:** 若 $X, Y$ 相互独立，则 $Cov(X, Y) = 0$ 
* **相关系数 (定义):**$$\rho_{XY} = \frac{Cov(X, Y)}{\sqrt{D(X) D(Y)}}$$
    **性质:** $-1 \le \rho_{XY} \le 1$。
* **n维正态分布的特性:** 服从 $n$ 维正态分布的随机变量的两两不相关性与相互独立性是**等价的**

### 第五章 大数定律与中心极限定理
## §1 大数定律 
* **一. 依概率收敛 (定义):** 设 $\{X_{n}\}$ 为随机变量序列, $X$ 为随机变量, 若任给 $\epsilon > 0$, 使得$$\lim_{n \to \infty} P\{|X_{n} - X| < \epsilon\} = 1$$则称 $\{X_{n}\}$ 依概率收敛于 $X$, 记为 $X_{n} \xrightarrow{P} X$
* **二. 常用的大数定律:**
    1.  **切比雪夫大数定律 :** 设 $\{X_{k}\}$ 为独立的随机变量序列, 且有相同的期望 $E(X_{k})=\mu$, 和方差 $\sigma^{2} > 0$, 则**样本均值** $\frac{1}{n} \sum_{k=1}^{n} X_{k}$ 依概率收敛于 $\mu$:$$\lim_{n \to \infty} P\{|\frac{1}{n} \sum_{k=1}^{n} X_{k} - \mu| [cite_start]< \epsilon\} = 1$$
    2.  **伯努利大数定律 (P100):** 设 $\mu_{n}$ 是 $n$ 次伯努利试验中事件 $A$ 发生的次数, $p$ 是 $A$ 发生的概率, 则**频率** $\frac{\mu_{n}}{n}$ 依概率收敛于**概率** $p$:$$\lim_{n \to \infty} P\{|\frac{\mu_{n}}{n} - p| [cite_start]< \epsilon\} = 1$$

## §2 中心极限定理 
* **林德伯格-列维定理 (独立同分布):**
    * 设 $X_{1}, X_{2}, \ldots, X_{n}$ 相互独立, 服从同一分布, 且 $E(X_{i}) = \mu$, $D(X_{i}) = \sigma^{2} > 0$。
    * 则当 $n$ 充分大时，**和的标准化变量**的分布函数趋于标准正态分布函数 $\Phi(y)$:$$\lim_{n \to \infty} P\{\frac{\sum_{i=1}^{n} X_{i} - n\mu}{\sqrt{n\sigma^{2}}} \le y\} = \Phi(y)$$

* **棣莫弗-拉普拉斯定理 (二项分布):**
    * 设 $\mu_{n} \sim B(n, p)$, 则当 $n$ 充分大时，**频数的标准化变量**的分布函数趋于标准正态分布函数 $\Phi(x)$:$$\lim_{n \to \infty} P\{\frac{\mu_{n} - n p}{\sqrt{n p (1-p)}} \le x\} = \Phi(x)$$

* **例 (保险公司亏本概率 - 应用):** 设 $X$ 表示一年内死亡人数, $X \sim B(n, p)$, $n=10000, p=0.006$。保险公司利润 $Y = 10000 \times 12 - 1000 X$。
    * **计算参数:** $E(X) = n p = 60$, $D(X) = n p (1-p) = 59.64$, $\sqrt{D(X)} \approx 7.7227$ 
    * **(1) 保险公司亏本的概率 $P\{Y < 0\}$ (推导):**$$P\{Y < 0\} = P\{120000 < 1000 X\} = P\{X > 120\}$$$$P\{X > 120\} = 1 - P\{X \le 120\}$$* **中心极限定理近似:** $\frac{X - E(X)}{\sqrt{D(X)}} \sim N(0, 1)$$$P\{X \le 120\} \approx \Phi(\frac{120 - 60}{7.7227}) \approx \Phi(7.75)$$$$P\{Y < 0\} \approx 1 - \Phi(7.75) \approx 0$$
