## 1.2 原子的核式结构
![[卢瑟福散射实验.png|500]]
$\color{blue}\text{Rutherford散射公式}$$$\color{red}d\sigma = \left( \frac{Z e^2}{4\pi\epsilon_0 \cdot M v^2} \right)^2 \frac{d\Omega}{\sin^4 \frac{\theta}{2}}$$
关键参数含义

| **符号**                    | **物理量**     | **说明**                                                                                                                                                                   |
| ------------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| $d\sigma$                 | **微分截面**    | 单位时间内散射到立体角 $d\Omega$ 内的粒子数与入射粒子通量之比。                                                                                                                                    |
| $Z$                       | **靶核的原子序数** | 靶核的电荷量为 $Q = Z e$。                                                                                                                                                       |
| $e$                       | **基本电荷**    | 入射粒子的电荷量 $q$ 在此公式的推导中通常设为 $e$ 或 $2e$（如 $\alpha$ 粒子），但您给出的公式形式是将靶核电荷 $Q=Ze$ 和入射粒子电荷 $q$ 统一写为 $Z e^2$ 的结构。更标准的形式通常是 $\frac{(Z e)(q)}{4\pi\epsilon_0}$ 的平方项，其中 $q$ 是入射粒子电荷。 |
| $\epsilon_0$              | **真空介电常数**  | 在 $\frac{1}{4\pi\epsilon_0}$ 中，构成国际单位制（SI）中的库仑常数 $k$。                                                                                                                    |
| $M$                       | **入射粒子的质量** | 有时用 $m$ 表示。                                                                                                                                                              |
| $v$                       | **入射粒子的速度** | $E = \frac{1}{2} M v^2$ 是入射粒子的动能。因此 $M v^2 = 2E$。                                                                                                                        |
| $\theta$                  | **散射角**     | 粒子偏离入射方向的角度。                                                                                                                                                             |
| $d\Omega$                 | **立体角元**    | 粒子被散射进入的立体角元素，$d\Omega = \sin\theta d\theta d\phi$。                                                                                                                      |
| $\sin^4 \frac{\theta}{2}$ | **角度依赖项**   | 表明截面随着散射角 $\theta$ 的减小而急剧增加（分母变小）。                                                                                                                                       |

$\color{blue}\text{最小接近距离公式}$$$\color{red}r_{m} = \frac{2Z e^2}{4\pi\epsilon_0 M v^2} \left( 1 + \frac{1}{\sin \frac{\theta}{2}} \right)$$
 参数含义
- $r_m$: **最小接近距离 (Closest Approach Distance)**。这是入射粒子在库仑散射过程中，其轨迹与散射中心（靶核）之间的最短距离。
- $Z e$: **靶核的电荷量**（$Z$ 是原子序数）。
- $e$: **基本电荷量**（假设入射粒子为 $e$，但对于 $\alpha$ 粒子，入射电荷应为 $2e$。您公式中的 $2Z e^2$ 结构通常暗示入射粒子为 $\alpha$ 粒子，其电荷为 $2e$，代入 $q=2e$ 后的结果）。
- $4\pi\epsilon_0$: **真空介电常数**的倒数 $\frac{1}{4\pi\epsilon_0}$ 是库仑常数 $k$。
- $M v^2$: **入射粒子动能的两倍** ($2E$)。
- $\theta$: **散射角**。
- $\sin(\theta/2)$: **角度依赖项**。

# 原子的能级和辐射 
## §2.1 光谱以及氢原子光谱
**一、光谱** 
1. **定义**：光谱是对某种电磁辐射的波长成分和强度分布的记录 。
2. **光谱类别** ：
    - **线状光谱**（原子光谱）
    - **带状光谱**（分子光谱）
    - **连续光谱**（热辐射）

**二、氢原子光谱** 

1. 巴耳末（Balmer）公式：描述可见光区氢光谱谱线的波长关系 $$\lambda=B\cdot\frac{n^{2}}{n^{2}-4},n=3,4,5,\cdot\cdot\cdot$$
    波长倒数（波数）表示为 ：$$\tilde{\nu}=\frac{1}{\lambda}=R_{H}(\frac{1}{2^{2}}-\frac{1}{n^{2}})$$
    其中 $R_H$ 称为**里德伯（Rydberg）常数** 。
2. **其他线系**：氢光谱线系都可以用一个简单公式表示 。 * **莱曼（Lyman）系**：$n=1, n'=2, 3, 4, ...$ (远紫外区) 
    - **巴耳末（Balmer）系**：$n=2, n'=3, 4, 5, ...$ (可见光区) 
    - **帕邢（Paschen）系**：$n=3, n'=4, 5, 6, ...$ (红外区) 
    - **布拉克（Brackett）系**：$n=4, n'=5, 6, 7, ...$ (远红外区) 
    - **普丰特（Pfund）系**：$n=5, n'=6, 7, 8, ...$ (超远红外区) 
3. 氢原子光谱的一般规律（里德伯公式） ：$$\tilde{\nu}=R_{H}(\frac{1}{n^{2}}-\frac{1}{{n^{\prime}}^{2}})=T(n)-T(n^{\prime})$$
    - 光谱为线状，谱线有确定位置 。
    - 谱线波数为**二光谱项** $T(n)$ 之差 。

三、经典理论对氢原子光谱的解释及困难 
根据卢瑟福（Rutherford）模型和经典电动力学：
1. **原子稳定性问题**：电子绕原子核运动会连续辐射能量，导致电子轨道半径逐渐缩小，最终原子会变成原子核大小，这与事实不符 。
2. 光谱线状问题：按电动力学，原子发光频率应等于电子运动频率，辐射时电子轨道连续缩小，发光频率应连续增大，而实际上光谱是线状的，理论与实际不符 。
    结论： 经典理论不能正确解释原子问题 。

## §2.2 经典理论的困难和光的波粒二象性

一、经典理论遇到的严重困难 
在进入20世纪后，经典理论在解释新的实验结果时遇到了困难，主要有：
1. **黑体辐射问题** 
2. **光电效应** 
3. **氢原子光谱** 。
**二、黑体辐射问题** 
4. **黑体**：吸收比（吸收的电磁能量与入射的电磁能量之比）为 1 的物体 。
5. **经典理论的解释（瑞利-金斯公式）**：在长波部分与实验符合，但在短波（紫外）部分趋于无穷大，导致著名的**紫外灾难** 。
    
6. **实验数据的总结（维恩公式）**：在短波部分与实验符合，但在长波部分明显不一致。
7. **普朗克（Planck）公式**：普朗克利用内插法，得到全频率范围内与实验结果相符的公式 。
    - **普朗克假设** ：
        - 黑体只能以**不连续**的 $\epsilon=h\nu$ **发射和吸收能量** 。
        - 能量单位 $E=h\nu$ 称为**能量子**，$h$ 称为**普朗克常数** 。
        - 原子振子的能量是**量子化**的，只能取 $h\nu$ 的整数倍 。
    - **量子思想的意义**：普朗克的量子思想是对经典理论的重大突破，指出了光的**微粒性** 。
**三、光电效应** 
8. **基本概念**：光照射到某些金属表面，有电子逸出的现象 。
9. **实验现象**：
    - 存在**临界频率** $v_0$，入射光频率小于 $v_0$ 时，不论光强多大，都没有光电子产生 。
    - 光电子的**动能与入射光的频率成线性关系** 。
    - 饱和光电流（光电子数目）与入射光强度成**正比** 。
    - 光电效应**瞬时发生** 。
10. **经典理论的困难**：无法解释临界频率、光电子能量与频率的线性关系，以及瞬时性 。
11. **爱因斯坦（Einstein）光量子理论** ：
    - **光量子（光子）**：爱因斯坦认为光不仅是电磁波，还是粒子，以能量 $E=h\nu$ 的微粒形式传播 。
    - 能量关系：光子的能量 $h\nu$ 被电子吸收，一部分用来克服金属表面对它的吸引（逸出功 $W$），另一部分是电子的动能。$$E_{kmax} = h\nu - W$$
    - **光的本质**：光具有**波粒二象性** 。
## §2.3 玻尔（Bohr）氢原子理论
为解决经典理论对氢原子光谱的困难 ，玻尔借助普朗克-爱因斯坦的量子化思想，提出了他的理论 。
1. **理论内容（基本假设）** ：
    - **定态假设**：原子能够而且只能够处于一系列分立的能量状态中，称为**定态**。处于定态时，原子不发出电磁辐射 。
    - 量子跃迁假设：原子在两个定态（设为 $E_n > E_{n'}$）之间跃迁时，才能吸收或发射电磁辐射，辐射的频率 $\nu$ 由下式决定 ：$$h\nu = E_n - E_{n'}$$
    - 轨道角动量量子化条件：原子处于定态时，电子绕原子核作轨道运动，轨道角动量 $L$ 满足量子化条件 ：$$\color{red}L = n\hbar \quad (n=1, 2, 3, \dots)$$
        （其中 $\hbar = \frac{h}{2\pi}$， $n$ 为主量子数）
2. **量子化的能级和轨道** ：
    - **能级公式**：玻尔理论导出的量子化的能级公式，与光谱项 $T(n)$ 对应 。 * **里德伯常数** $R_H$：玻尔理论计算出的里德伯常数值与实验测得值高度吻合 。
3. **重要意义和局限性** ：
    - **重要意义**：首次打开了认识原子结构的大门，为量子力学的建立起到了重要的引导作用 。
    - **局限性**：
        - 不能解释较复杂原子（如氦原子）的光谱 。
        - 不能给出光谱的谱线强度（相对强度） 。
        - 量子化概念的物理本质不清楚 。
## §2.4 类氢体系光谱
1. **类氢体系**：指原子核与氢原子不同，但核外仍只有一个电子的类原子体系，如 $\text{He}^+$, $\text{Li}^{++}$ 等 。
2. **毕克林（Pickering）系**：后被发现是 $\text{He}^+$ 光谱，可应用玻尔理论解释 。
3. **里德伯常数的差异**：类氢体系与氢原子光谱的差异是由于**里德伯常数不同**造成的，这是因为玻尔理论最初假设了**原子核不动** 。
4. **考虑原子核运动**：考虑电子与原子核绕质心运动，引入**折合质量** $\mu$ 代替电子质量 $m$ 。
5. 类氢体系光谱的一般表示 ：$$\tilde{\nu}=R_{Z}(\frac{1}{n^{2}}-\frac{1}{{n^{\prime}}^{2}})$$
    其中 $R_Z$ 是考虑了原子核运动的里德伯常数 。

## §2.5 弗兰克-赫兹实验（Franck-Hertz Experiment）

**主要目的**：通过实验方法支持原子**能级**和电子运动**轨道**的**量子化**特点 。
1. **实验结论**：电子加速能量达到某一特定值（**激发电势**）时，电流突然下降，表明电子与原子发生**非弹性碰撞**，能量被原子全部吸收，使原子从低能级**跃迁**到高能级，支持了原子能级的**分立性** 。
2. **电离电势**：使原子中电子离去所需的电压。电离电势可由原子光谱的线系限推算 。
## §2.6 $\color{red}\text{量子化通则}$
1. 索末菲（Sommerfeld）量子化通则 ：
 $$\color{red}\oint p_q dq = n_q h$$
    其中 $q$ 为广义位移，$p$ 为对应的广义动量，积分限为一个周期 。
2. **谐振子的量子化**：利用索末菲量子化通则，可推出谐振子的能量也是**量子化**的，即 **$E = n h\nu$**，与普朗克的能量子假设一致 。

## §2.7 电子的椭圆轨道
1. **运动轨道**：总能量 $E < 0$ 时，电子绕原子核做**椭圆运动** 。
2. **量子化椭圆轨道**：利用索末菲量子化条件，可得到电子的运动轨道应为**量子化的椭圆轨道** 。 * **量子数**：轨道和两个量子数 $n$（**主量子数**）和 $n_\phi$（**角量子数**）有关 。
    - **能级**：体系能量只决定于**主量子数 $n$**，与 $n_\phi$ 无关 。
    - **简并态**：对于某一 $n$ 值，有 $n$ 个可能轨道，能量相同，称为**简并态** 。
3. **相对论效应**：
    - 导致轨道不闭合，像椭圆轨道有一个**连续的进动** 。
    - 使得 $n$ 相同而 $n_\phi$ 不同的轨道，能量略有差异，导致**简并态解除**（能级分裂） 。
    - 引起原子光谱的**精细结构** 。
## §2.8 史特恩-盖拉赫实验和原子空间取向的量子化

1. **史特恩-盖拉赫（Stern-Gerlach）实验**：通过测量原子在不均匀磁场中的行为，证明了原子在磁场中**空间取向的量子化**（角动量在磁场方向分量的量子化） 82。2. **轨道磁矩**：电子绕核运动产生**轨道磁矩** $\mu_\phi$ 。
    - $\mu_\phi$ 与轨道角动量 $L$ **方向相反**，且 $\mu_\phi$ 也是**量子化**的 。
    - $\mu_B = \frac{e\hbar}{2m}$ 称为**玻尔磁子**，是轨道磁矩的最小单位 。
2. **空间量子化**：在磁场作用下，电子的轨道及轨道角动量的**取向**（在磁场方向的分量）也是**量子化**的 。
    - 角动量在磁场方向的分量 $L_\psi$ 满足 $L_\psi = n_\psi \hbar$ 。
    - 轨道取向量子化，即轨道角动量在磁场或电场方向的分量是量子化的 。



# 碱金属原子

## **一、碱金属原子的光谱和能级 (§1)**

1. **研究对象:** 碱金属元素（锂 Li、钠 Na、钾 K、铷 Rb、铯 Cs、钫 Fr），它们在周期表中属同一族，性质相似 。它们的原子结构比类氢体系复杂，但相对简单 。
2. **光谱特点**
    - **谱线系:** 碱金属原子的光谱分为一系列线系，最常见的有四个：
        - **主线系 (Principal Series):** 主要在紫外区 (锂的第一条在红色) 。
        - **第一辅线系/漫线系 (Diffuse Series):** 在可见光区 。
        - **第二辅线系/锐线系 (Sharp Series):** 在可见光区 (锂的第一条在红外) 。
        - **柏格曼系/基线系 (Fundamental Series):** 在红外区 。
    - **$\color{red}谱线公式$:** $\color{red}\tilde{v}=\tilde{v}_{\infty}-R/n^{*2}$ 。
        - $n^{*}$ 称为**有效量子数**，一般不为整数 。
        - $\Delta=n-n^{*}$ 。
    - **有效量子数 $n^{*}$ 的规律 (角量子数 $l=0, 1, 2, 3$ 对应 s, p, d, f 电子态) ：
        - $n^{*}$ 都比主量子数 $n$ 略小或相等，可表示为 $n^{*}=n-\Delta$ 。
        - 同一线系的 $\Delta$ 值近似相同 。
        - **光谱项**可表示为 $T=\frac{R}{n^{*2}}=\frac{R}{(n-\Delta_{l})^{2}}$ 。
3. **波数公式 (以锂原子为例)** ：
    - **第二辅线系 ($s$ 态, $l=0$):** $\tilde{v}_{n}=\frac{R}{(2-\Delta_{p})^{2}}-\frac{R}{(n-\Delta_{s})^{2}}, n=2,3,4,...$
    - **主线系 ($p$ 态, $l=1$):** $\tilde{v}_{n}=\frac{R}{(2-\Delta_{s})^{2}}-\frac{R}{(n-\Delta_{p})^{2}}, n=3,4,5,...$
    - **第一辅线系 ($d$ 态, $l=2$):** $\tilde{\mathcal{V}}_{n}=\frac{R}{(2-\Delta_{p})^{2}}-\frac{R}{(n-\Delta_{d})^{2}}, n=3,4,5,...$
    - **柏格曼系 ($f$ 态, $l=3$):** $\tilde{v}_{n}=\frac{R}{(3-\Delta_{d})^{2}}-\frac{R}{(n-\Delta_{f})^{2}}, n=4,5,6,...$


4. **能级**
    - **$\color{red}能级公式$:** $\color{red}E_{nl}=-hcT=-\frac{hcR}{(n-\Delta_{l})^{2}}=-hcR\frac{Z^{*2}}{n^{2}}$, 其中 $Z^{*}=\frac{n}{n-\Delta_{l}}\ge1$ 是**有效电荷数** 。
    - **特点:** 碱金属原子的能级不仅与主量子数 $n$ 有关，还与角量子数 $l$ 有关 。对于同一 $n$，碱金属能级都比氢原子（H）能级低 。

5. **碱金属的结构特点**
    - 碱金属原子由**原子实**和**价电子**构成 。
    - **原子实:** 由原子核和内层电子（占据内层轨道）组成，结构稳定 。
    - **价电子:** 处于原子最外层，占据原子实以外的轨道，与原子实的结合不牢固，容易脱离和激发，原子的性质主要由它决定 。
    - **有效电荷:** 原子实带有 $+e$ 单位正电荷，价电子在其 Coulomb 场中运动，结构类似氢原子 。


6. **影响能级的因素** (解释与氢原子能级的差异) 
    - **原子实的极化:** 价电子对原子实的作用使原子核与负电荷中心偏移，原子实极化形成电偶极子，反作用于电子使其能量降低 。
        - $l$ 值越小，椭圆轨道偏心率越大，极化越强，$\Delta$ 越大，能量降低越多 。
    - **轨道的贯穿:** $s$ 和 $p$ 态对应偏心率很大的轨道，价电子可能穿入原子实，形成**轨道贯穿效应** 。
        - 在原子实外时，$Z^{*}=1$；贯穿原子实时，$Z^{*}>1$ 。
        - 贯穿效应使能级比氢原子能级低得多 
## **二、碱金属原子光谱的精细结构 (§2)**

1. **定义:** 用高分辨率光谱仪观察，碱金属原子的光谱线由更细的两条或三条谱线组成，称为**光谱精细结构** 。
2. **线系特点** ：
    - **主线系 ($p \to s$):** 每条谱线皆为**双线**，双线间隔逐渐减小 。
    - **第二辅线系 ($s \to p$):** 由**双线**组成，双线间隔**固定** 。
    - **第一辅线系 ($d \to p$):** 由**三线**组成，最外两条间隔固定且与二辅系相同 。
3. **能级特点 (推测):**
    - $s$ 能级为**单层** 。
    - $p, d, f$ 等能级都为**双层** 。
    - 对同一 $l$ 值，双层间隔随 $n$ 的增大而**减小** 。
    - 对同一 $n$ 值，双层间隔随 $l$ 的增大而**减小** 。
4. **精细结构的解释:** 光谱精细结构和斯特恩-格拉赫 (Stern-Gerlach) 实验的结论表明，原子中存在一种运动，其力学量在某一方向的取值只有 2 个值，这种运动就是**电子自旋** 。
    
## **三、电子自旋与轨道运动的相互作用 (§3)**
1. 电子的自旋假设 (Uhlenbeck 和 Goudsmit 提出):
    - 电子具有**自旋角动量** $\vec{s}_{p}$，在空间任何方向上的投影只能取两个值 。
    - 电子具有**自旋磁矩** $\vec{\mu}_{s}$，在空间任何方向上的投影只能取两个值 。
2. **自旋轨道作用的实质:** 磁相互作用 。
    - 电子的自旋磁矩 $\vec{\mu}_{s}$ 与核电荷绕电子旋转产生的磁场 $\vec{B}$ 作用，产生**自旋附加能量** $\Delta E_{so}$，导致能级形成双层结构 。
3. **原子态的表示:**
    - 碱金属原子状态由价电子的**轨道角动量** $(l)$、**自旋角动量** $(s)$ 和**总角动量** $(j)$ 决定 。
    - 原子态符号为 $n^{2s+1}L_j$ ：
        - $n$: 主量子数 。
        - $2s+1$: **能级层数** (自旋量子数 $s$ 决定，碱金属 $s=1/2$, $2s+1=2$) 。
        - $L$: 轨道角量子数 $l$ 对应的字母 ($l=0 \to S, l=1 \to P, l=2 \to D, \dots$) 。
        - $j$: 总角动量量子数 $j = l \pm 1/2$ 。
    - 碱金属原子态属于**双层系统** (例如 ${}^2S_{1/2}, {}^2P_{1/2}, {}^2P_{3/2}, \dots$) 。
![[碱金属原子态符号.png|500]]
## **四、单电子辐射跃迁的选择定则 (§4)**
1. **选择定则:** 只有满足一定条件的跃迁才会发生 。
    - **轨道角量子数:** $\Delta l = \pm 1$ 。
    - **总角动量量子数:** $\Delta j = 0, \pm 1$ (但 $j=0 \to j=0$ 的跃迁不发生) 。
2. **实际光谱:** 碱金属光谱的精细结构符合选择定则 。
    - **主线系** ($^2P_{1/2, 3/2} \to {}^2S_{1/2}$): **双线结构** 。
    - **锐线系/第二辅线系** ($^2S_{1/2} \to {}^2P_{1/2, 3/2}$): **双线结构** 。
    - **漫线系/第一辅线系** ($^2D_{3/2, 5/2} \to {}^2P_{1/2, 3/2}$): **三线结构** 。
## **五、氢原子光谱的精细结构与蓝姆移动 (§5)**
1. **精细结构能级:** 氢原子和碱金属原子的能级由三部分组成 ：
    - 主要部分 (Bohr 理论) 。
    - 相对论效应的附加能量 。
    - 电子自旋的附加能量 。
2. **能级形式:** $\boldsymbol{E_{n} \to E_{n, j}}$，原能级上下分裂为**双层能级** 。
3. **氢原子能级特点:** 有效电荷数 $Z-\sigma=Z-s=1$，能级只与 $n$ 和 $j$ 有关 。
    - 存在**能级简并**：同一 $j$ 值，一般对应 2 个 $l$ 值，能量相同。例如 ${}^2S_{1/2}$ 态和 ${}^2P_{1/2}$ 态能量相同 。
4. **巴耳末系第一条谱线 (3 → 2) 的精细结构:**
    - 可能跃迁有 7 种，由于能级简并，形成**5 条**谱线 。
    - 实验上一般只能分辨出**双线** 。
5. **蓝姆移动 (Lamb Shift)**
    - 实验测得巴耳末系谱线的双线间隔比理论值小约$0.011 \text{cm}^{-1}$
    - **蓝姆和李瑟福 (Lamb and Retherford)** 在 1947 年测得氢原子 ${}^2S_{1/2}$ 态比 ${}^2P_{1/2}$ 态**高出** $1058 \text{MHz}$ (即 $0.03 \text{cm}^{-1}$) 。
    - 这一发现 (蓝姆移动) 解释了精细结构实验观察值与理论值的差异 。


# 多电子原子 
## §1 氦及第二族元素的光谱和能级
- **研究对象：** 氦（He）及第二族元素（碱土族，如Mg、Ca、Sr等），它们都具有**两个价电子** 。
- **光谱特点：** 都具有**两套线系** 。
    - 一套为**单线结构**（单重态能级间的跃迁产生） 。
    - 另一套为**复杂的多线结构**（三重态能级间的跃迁产生） 。
- **能级特点：** 相应地，能级也分为**单重态能级**（单层）和**三重态能级**（三层）两套 。
- **跃迁规则：** **两套能级之间没有跃迁** 。
- **He的能级：** 能级分单态和三重态。例如，氦的一条重要谱线$587.6 \text{nm}$对应三重态跃迁的漫线系第一条谱线（$3D \to 3P$） 。

---

### §2 两个价电子的原子态

- **原子态决定因素：** 原子由原子实和价电子组成，其运动状态和性质主要由**价电子**决定 。
- **电子组态：** 原子中价电子的各种状态组合 。例如：$\text{He}$的基态是$1s1s$，第一激发态是$1s2s$ 。
- **影响原子态的相互作用：** 两个价电子的**自旋**和**轨道**运动相互作用形成原子态 。主要有六种相互作用，其中 $G_1(s_1, s_2)$, $G_2(l_1, l_2)$, $G_3(l_1, s_1)$, $G_4(l_2, s_2)$ 较强 。
- **耦合类型：**

| **耦合类型**  | **适用原子**                       | **相对强度**                | **合成步骤**                                                                                                                                                                                                    |
| --------- | ------------------------------ | ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **LS 耦合** | 轻原子（如$\text{He}$, $\text{Mg}$） | $G_1, G_2 \gg G_3, G_4$ | **第一步：** $\vec{P}_{s1} + \vec{P}_{s2} \to \vec{P}_S$（总自旋角动量） <br>**第二步：** $\vec{P}_{l1} + \vec{P}_{l2} \to \vec{P}_L$（总轨道角动量） <br>**第三步：** $\vec{P}_S + \vec{P}_L \to \vec{P}_J$（总角动量，原子态）                  |
| **jj 耦合** | 重原子                            | $G_3, G_4 \gg G_1, G_2$ | **第一步：** $\vec{P}_{l1} + \vec{P}_{s1} \to \vec{P}_{j1}$（电子1总角动量） <br>**第二步：** $\vec{P}_{l2} + \vec{P}_{s2} \to \vec{P}_{j2}$（电子2总角动量）  <br>**第三步：** $\vec{P}_{j1} + \vec{P}_{j2} \to \vec{P}_J$（体系总角动量，原子态） |


- **$\color{red}LS角动量合成法则$：**
    - **总轨道角动量$L$：** $L = l_1 + l_2, l_1 + l_2 - 1, \dots, |l_1 - l_2|$ 。
    - **总自旋角动量$S$：** 对于两电子系统， $S=1$（三重态）或$S=0$（单重态） 。
    - **总角动量$J$：** $J = L + S, L + S - 1, \dots, |L - S|$ 。

- $\color{red}jj角动量合成法则$
	- $j_1=l_1+s,l_1-s$
	- $j_2=l_2+s,l_2-s$
	- $J=j_1+j_2,\dots,|j_1-j_2|$

## §3 $\color{red}泡利原理与同科电子$

- **泡利不相容原理（Pauli Exclusion Principle）：** 同一原子中，不能有两个电子处于**完全相同的状态**，即其四个量子数（$n, l, m_l, m_s$）不能完全相同 。
    - **应用：** 解释了$\text{He}$原子基态（$1s^2$）只能存在**单重态**（$S=0$），不存在三重态（$S=1$）。因为$n, l, m_l$均相同，则$m_s$必须相反，导致总自旋量子数$S$只能为$0$ 。
- **同科电子：** 主量子数$n$和角量子数$l$都相同的电子 。
    - 同科电子由于泡利原理的限制，形成的原子态数目要**少于**非同科电子组态 。
- **洪特定则（Hund's Rules）：** 用于确定能级顺序（最低能级） 。
    1. **最大重数：** $S$最大（重数$2S+1$最大）的能级位置最低。
    2. **最大轨道：** 在重数相同的能级中，$L$最大的能级位置最低。
    3. **最低$J$值：** 对于形成的精细结构能级：
        - 当电子层**半满或未满**时，$J$最小的能级位置最低（正常多重态）。
        - 当电子层**半满以上**时，$J$最大的能级位置最低（反常多重态）。
- **朗德间隔定则（Landé Interval Rule）：** 在一个多重能级结构中，相邻的两能级间隔（$E_J - E_{J-1}$）与两个$J$值中**较大的成正比** 。

## §4 复杂原子光谱的一般规律
- **光谱和能级的位移律：** 原子序数$Z$的中性原子，与原子序数$Z+1$的原子的一次电离离子，具有**类似的光谱和能级结构**。因为它们的价电子数目相同 。
- **多重性的交替律：** 元素交替地具有**偶数或奇数的多重态** 。
    - 重数$2S+1$取决于价电子数目。
    - 奇数个价电子 $\to$ 偶数多重态（双重、四重等） 。
    - 偶数个价电子 $\to$ 奇数多重态（单重、三重等） 。

---

## §5 辐射跃迁的普适选择定则
- **基本原则：** 跃迁只能发生在**不同宇称**的状态之间，即原子体系发生跃迁时，其宇称**一定改变** 。

## §6 原子激发和辐射跃迁的实例——He-Ne激光器

- **基本结构：** $\text{He}$、$\text{Ne}$气体密封在玻璃管中，施加高压电场 。
- 工作原理： 1. 高能电子激发 $\text{He}$ 原子到亚稳态（$\text{He}$ $1s2s$的$^1S_0$和$^3S_1$能级）。
    2\. $\text{He}$ 亚稳态能级与 $\text{Ne}$ 的 $2p^55s$ 和 $2p^54s$ 能级高度接近 。
    3. $\text{He}$ 亚稳态原子与基态 $\text{Ne}$ 原子碰撞，发生能量共振转移，将 $\text{Ne}$ 原子激发到上能级 。
    4. 在 $\text{Ne}$ 的上能级（$2p^55s$ 和 $2p^54s$）集中了大量原子，形成粒子数反转，发生受激辐射，输出激光 。
