# 第七章 原子的壳层结构 复习笔记

## **§1 元素性质的周期性**

1. **发现历史**
    - 1869年，门捷列夫发现元素性质按原子量次序排列显示出周期性 。
    - 后来的研究表明，元素的性质与**原子序数**（即**核的电荷数**）的排列更吻合 。
2. **元素周期表的基本特点**
    - **周期（行）：** 周期数不同，所包含的元素数目不完全相同（依次为 2, 8, 8, 18, 18, 32, 17+ (未满)） 。
    - **族（列）：** 包括8个A族、过渡元素（如VIB族包含3列）、以及B族等 。
3. **周期性表现**
    - 元素的物理性质和化学性质的变化显示出周期性，这实质上是**原子结构周期性**的反映 。
    - **例：** 元素的第一电离能随原子序数的变化呈现明显的周期性 
## **§2 原子的电子壳层结构**
1. **原子中的电子状态**
    - 原子中电子的运动状态由四个量子数反映（自旋量子数 $s=1/2$ 为常数） 。
        
    - **主量子数 ($n$):** $1, 2, 3, \ldots$。反映电子运动区域的大小和能量的主要部分（即轨道大小） 。
        
    - **轨道角动量量子数 ($l$):** $0, 1, 2, \ldots, n-1$。反映轨道的形状和轨道角动量的大小，对电子能量也有影响 。
        
    - **轨道方向量子数 ($m_l$):** $l, l-1, \ldots, 0, \ldots, -l$。反映轨道在空间（如磁场方向）的取向 。
        
    - **自旋方向量子数 ($m_s$):** $+1/2, -1/2$。反映电子自旋的取向 。
        
    - **任一量子数变化，电子的运动状态即发生改变** 。
        
2. **Pauli原理 (Pauli Exclusion Principle)**
    
    - 原子中**任意两个电子的状态不能完全相同** 。
        
    - 即，反映电子运动状态的四个量子数 ($n, l, m_l, m_s$) 不能完全相同 。
        
3. **原子壳层划分**
    
    - **主壳层（壳层）：** $n$ 相同的电子构成，它们在同样大小的区域运动，能量的主要部分相同 。
        
    - **次壳层：** $n$ 相同但 $l$ 不同的电子构成，它们轨道的形状不同，能量有微小差异 。
        
4. **最大电子数（磁场中）**
    
    - 次壳层可容纳的最大电子数（$N_l$）：$$N_{l}=2(2l+1)$$
        
        - （因 $m_l$ 有 $2l+1$ 个值， $m_s$ 有 2 个值） 。
            
    - 主壳层可容纳的最大电子数（$N_n$）：$$N_{n}=\sum_{l=0}^{n-1} 2(2l+1) = 2n^2$$
        - （将 $l=0$ 到 $n-1$ 的所有次壳层电子数相加） 。

## **§3 原子基态的电子组态**
1. **基态**
    - 指原子能级**最低**的状态 。
    - 电子在填充原子态时要遵从**Pauli原理**和**其他排布规律**（如使 $S$ 最大） 。
2. **能级填充顺序（实际填充）**
    - 电子按周期表顺序逐个填充，并尽可能处于各自可能最低的能级中 。
    - **重要例外：** 第四周期开始，元素的电子并非严格按照主量子数 $n$ 依次填充 。
    - **能级高低：** 实验发现 $4s$ 能级比 $3d$ 能级低，因此电子先填充 $4s$ 轨道，后填充 $3d$ 轨道 。
3. **基态原子态的得出**
    - 确定基态时，应运用 Pauli原理以及能级规律（如 **$L$ 和 $S$ 越大，能级越低**，以及洪特规则等），找出原子的基态 。
    - **简便法：** 计算 $L$ 和 $S$，令 $J=|L-S|$，将电子（自旋取向）按右图填充 $m_l$ 各值，可保证 $S$ 最大 。
